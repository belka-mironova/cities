### Project setup

1. Clone the repo
   ```sh
   git clone https://github.com/bella-mir/cities.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the frontend part of the application in a development mode
   ```sh
   npm run start
   ```

## Stack:

Front-end:

- JavaScript
- React.js
- Redux Toolkit
- Map GL
